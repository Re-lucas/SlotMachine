# Slot Machine Game

This is a simple slot machine game implemented in Python using the Tkinter library for the graphical user interface.

## How to Run

Make sure you have Python installed on your system. Open a terminal and run the following command:

```bash
python slot_machine.py
```

## Game Rules

- The player starts with 100 credits.
- Place a bet by clicking on the bet buttons (1, 2, 5, 10).
- Click the "Spin" button to spin the slot machine.
- Hold buttons allow the player to keep a slot's value for the next spin.
- Winning combinations result in credit prizes.
- If the player gets three Jackpots, they win the accumulated jackpot amount.

## User Interface

- **Credits Label:** Displays the current number of credits.
- **Jackpot Label:** Shows the current jackpot amount.
- **Bet Label:** Indicates where to place your bet.
- **Bet Buttons:** Choose the bet amount.
- **Spin Button:** Spin the slot machine.
- **Hold Buttons:** Allow the player to hold specific slots.
- **Result Labels:** Display the results of each slot spin.

## How to Play

1. Enter your bet by clicking on the bet buttons.
2. Click the "Spin" button to start the game.
3. View the results in the result labels.
4. Collect your winnings if you get a winning combination.
5. Repeat or quit the game based on your credits and choice.
