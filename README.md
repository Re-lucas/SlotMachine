## Introduction

This project is a basic slot machine game developed with Python's Tkinter library. It allows players to place bets, spin the slots, and try their luck to win credits.

## Features

- User-friendly graphical interface.
- Betting options: 1, 2, 5, and 10 credits.
- Hold feature for slots.
- Random slot results based on predefined symbols.
- Jackpot system.
- Game statistics (games won, games lost).

## Requirements

- Python 3.x
- Tkinter (usually included with Python installations)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Re-lucas/SlotMachine.git
```

2. Navigate to the project directory:

```bash
cd slot-machine-game
```

3. Run the game:

```bash
python slot_machine.py
```

## How to Play

1. Launch the game using the provided installation instructions.
2. Place a bet by clicking on the bet buttons (1, 2, 5, 10).
3. Click the "Spin" button to spin the slots.
4. Optionally, use the "Hold" buttons to keep specific slots for the next spin.
5. Follow the game's feedback messages and enjoy playing!

## Game Rules

- **Winning Combinations:**
  - Cherry in the first slot: Player gets their bet back.
  - Cherry in the first two slots: 2x player’s bet.
  - Three cherries: 3x player’s bet.
  - Three Lemons: 5x player’s bet.
  - Three Lucky 7’s: 7x player’s bet.
  - Three Bars: 10x player’s bet.
  - Three Diamonds: 20x player’s bet.
  - Three Jackpots: All the credits in the jackpot.

- **Jackpot:**
  - Jackpot credits increase with each game won.
